### Data Management
- Expect lots of Data!
- To help manage your data efficiently:
1. Back Up Locally
   - Immediately back up your data to a personal hard drive (USB 3.0 recommended).
   - Please do not leave data transferring on the Snouty when other users are scheduled.
   - Once your backup is complete, immediately remove your data from the computer.


    OR 


1. Upload to the CALM file server 
   - The Snouty is directly connected to ~petabyte server. Instructions to access the server can be found [here](../data-analysis/Data-Storage-and-Compute.html).
   - The server offers a 3-month grace period, this is not permanent storage.
   - You can access the server on the UCSF network or via VPN.
   - The server is only available to internal users.
   - Pro Tip: The CALM offers analysis workstations with 10 Gb/s high-speed access to the mic.ucsf.edu file server. These can be booked through iLabs for data transfer, viewing, or analysis, and can be operated remotely. To get remote access first e-mail Kari, then instructions for remote workstation access can be found [here](../data-analysis/Remote-Access-to-the-Workstations.html).
