## Easy Install clear volume

1.  Update Fiji
 1. Help → Update
 2. [
2.   Once you get fiji to an update point where it will open the window
 with the \"Manage update sites\" in the corner, click \"Manage
 update sites\"
 1. [
 2. Check the box for \"Clear volume,\" then \"Apply and Close\"
 3. [
 4. Apply changes
3. Finish updating and restart

## Rendering issue- FIX- Note you should just do after an install

This was an issue found by Tanner Fadero and fixed by Nico Stuurman
[https://forum.image.sc/t/clearvolume-rendering-issue/72381](https://forum.image.sc/t/clearvolume-rendering-issue/72381)

1. [Go to the Fiji folder → Jars folder and find the  clearvolume.jar
 file ]
2. [Replace it with the
 [clearvolume-1.0-SNAPSHOT.jar](https://forum.image.sc/uploads/short-url/sQ5spHwMw66xI8Y98uQlC3V9jHm.jar) ]
3. [clearvolume-1.0-SNAPSHOT.jar](attachments/701467993/701468001.jar)
