## Easy Install clear volume

1.  Update Fiji
2. Help → Update
3. Once you get fiji to an update point where it will open the window
 with the "Manage update sites" in the corner, click "Manage
 update sites"
4. Check the box for "Clear volume," then "Apply and Close"
5. Apply changes
6. Finish updating and restart
