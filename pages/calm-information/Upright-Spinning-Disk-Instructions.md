# Upright Spinning Disk Instructions

CAMM upright

## To Turn on the System For Spinning Disk

1. First Press the Master Power on the control tower (below the Computer)
2. Press the "On" switch on the Black Power Bar to the right of the encloser. (On the air table)
3. Toggle the Keys to the Laser stacks behind the microscope.
   1. They should be to the right/perpendicular to the ground position. Turn up right, then back to the right/perpendicular to the ground position.
   2. ![Laser key position](attachments/654283421/654283423.png)

## For Spinning Disk

### Open Software and Save file

1. Open software: Desktop icon version Slidebook 2023- Confocal
2. ![Slidebook icon](attachments/654283421/654283425.png)
   1. Save Slide file
   2. This will be the overall file that contains all images
   3. A "New Slide" opens when the software opens
3. To save this Go to File > Save As Then Save Under DATA(F:)/Data/\[your folder\]

### The Focus Window

![Focus window](attachments/654283421/654283426.png)

In this window you will:

- Change light path
- Program settings
- See histogram
- Set up tiling
- Set up Z

### Changing Light path -> turning on widefield

Go to Filter Set -> Select Widefield

![Widefield selection](attachments/654283421/654283427.png)

Then choose the color/filter and "open Fluor to the right to see the channel"

![Filter channel selection](attachments/654283421/654283428.png)

To see by Eye there is a push Bar on the physical microscope, push in to see by eye, pull out to see on the computer.

**REMEMBER TO PULL THE PUSH BAR (to the right of the eye pieces) OUT WHEN GOING TO CONFOCAL**

### Changing Light path -> turning on Confocal

Under the filter set **Select confocal**

![Confocal selection](attachments/654283421/654283430.png)

Click on Filter/Channel that you want to image

Then Click on Live button (shows as stop in the picture)

"Open Alt" should change to "Close Alt"

![Live button](attachments/654283421/654283431.png)

Image should appear on screen

To change laser Power (lower right in focus window) or Exposure (above the filter set in focus window) you can adjust these in this window, HOWEVER, they are NOT automatically remembered per channel.

- To be separated Per channel that needs to be done in the Capture window

![Capture window settings](attachments/654283421/654283432.png)

In the capture window, select the wavelength then set exposure and then the current/laser power

![Wavelength and power settings](attachments/654283421/654283433.png)

### Histogram

Is in the focus window under the "Camera" tab

![Histogram](attachments/654283421/654283434.png)

### Setting Z stacks

Select 3D in Capture type in Capture window

Then under 3D Capture:

- If you want Relative Z choose:
  - Current for where you are currently
  - Reference will Call on XY tab
  - NOTE - for both of these remember to set the range or number of planes to the right side

- If you want Absolute choose use top and bottom:
  - Make sure to set top and bottom in focus window

Absolute/top and Bottom Z in Focus Window:

- Click on the Z tab in the focus window
- Use the Analog Voltage Z stage to move up and down
- Set top and bottom with the buttons

![Z stack settings](attachments/654283421/654283437.png)
