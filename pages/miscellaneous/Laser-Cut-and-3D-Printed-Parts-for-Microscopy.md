# Laser Cut and 3D Printed Parts for Microscopy

Here are 3D parts we have designed for microscopes. Most have been designed in either Solidworks or Autocad Inventor ([which can be obtained free for academic use](http://www.autodesk.com/education/schools-educators-software)).

## Nikon Correction Collar Motor

These are parts for a correction collar that clamps on to a Nikon 40x / 1.15 WI objective and allows the collar to be driven by a servo motor.

| Name | Description | STL File | CAD File |
|---|---|---|---|
